Draft 3-part agenda (best-practice format)

Why this works: low-pressure welcome, main event, optional family-friendly closeout.

Friday Night — Welcome Mixer

7:00–10:00 PM • Casual bar/rooftop in Lexington

Name tags, throwback playlist, slideshow wall, first reconnects.

Saturday — Main Reunion Event

6:00–11:00 PM • Signature venue + dinner + remarks

Class memory reel, “where are we now” 90-second share-outs, teachers/coaches shoutouts, photo booth.

Sunday — Family Brunch / School Walkthrough

10:30 AM–12:30 PM • Optional

Great for people traveling with spouses/kids and one last goodbye window.

Venue shortlist (v1.1)

  • The Campbell House — classic Lexington feel, good for ballroom-style main event
  • Hilton Lexington/Downtown — central location, easy hotel logistics for out-of-town classmates
  • Marriott Griffin Gate — large-capacity option with parking + room block potential
  • 21c Museum Hotel — modern vibe for Friday mixer + photo-heavy atmosphere
  • The Mane on Main — rooftop/social setting for casual kickoff

Selection criteria: central location, parking/hotel access, AV support, flexible food-and-beverage minimums, and clear weather backup.
